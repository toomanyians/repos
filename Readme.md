# Repository Information (Why we call it Readme.txt)

The primary reason for this repository is that I cannot share code produced for my employer. Intellectual property rights and confidentiality agreements prevent me from publishing that work.

As a result, these projects provide real-world examples that demonstrate my skills and experience using my own data, tools, and business requirements.

I hope sharing this knowledge is useful to others.

---

## Diabetes Project

**./Diabetes**  
This project focuses on using Power BI for data visualization.

I monitor my cat's blood glucose levels and report them to a central tracking system. The collected data is used to identify trends that help guide insulin dosage adjustments and optimize treatment decisions.

### Folders

- **./Diabetes/Data** — Data stored in compressed UTF‑8 encoded CSV files  
- **./Diabetes/Documents** — Project documentation and reference material  
- **./Diabetes/Power BI** — Dashboards built using Power Query (M Language) and DAX

## EveOnline Project

**./EveOnline**  
This project focuses on collecting, processing, and analyzing data from the EVE Online ESI API.

REST API calls can come in many forms, with multiple issues and limitations. This is not a key to all the techniques, but it does handle a lot of variations.

### Folders

- **./EveOnline/Data** — Data stored in compressed UTF‑8 encoded CSV files  
- **./EveOnline/Documents** — Project documentation and reference material  
- **./EveOnline/Python** — Python scripts for OAuth authentication, API retrieval, and automation  
- **./EveOnline/Powershell** — PowerShell scripts for OAuth authentication, API retrieval, and automation  
- **./EveOnline/Models** — Excel and OpenOffice data models and supporting analysis artifacts

## Intune Lab

**./Intune Lab**  
This project focuses on submitting, collecting, processing, and analyzing data from various Azure APIs.

As robust as Intune is, it still has reporting and inventory limitations. Extending Intune's functionality with compliance reporting is the focus of this lab.

### Folders

- **./Intune Lab/OOBE Image** — Documentation for lab test device builds
- **./Intune Lab/Projects** — Project documentation and reference material
